# 🚀 Nehil's Premium Developer Showcase

![Portfolio Cover](./public/cover.png)

## ✨ Overview
A cutting-edge, futuristic personal developer portfolio built with modern web technologies. This project features high-end animations, 3D interactive elements, and a sleek glassmorphism UI/UX design.

## 🛠️ Tech Stack
- **Frontend Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: 
  - [GSAP](https://greensock.com/gsap/) (ScrollTrigger, Text Splitting)
  - [Framer Motion](https://www.framer.com/motion/) (Spring physics, 3D transforms)
- **3D Graphics**: [Three.js](https://threejs.org/) & [React Three Fiber](https://github.com/pmndrs/react-three-fiber)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Forms & Validation**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)

## 🌟 Key Features
- 🖱️ **Magnetic Cursor**: Custom spring-physics cursor that reacts to interactive elements.
- 🌌 **3D Particle Background**: Immersive Three.js environment with 1000+ interactive particles.
- 🎯 **Advanced Hero Section**: 3D tilt effects, floating decorative elements, and GSAP text reveals.
- 📊 **Animated Stats**: Real-time counter animations for achievements and skills.
- 🎨 **Glassmorphism UI**: High-end aesthetic with frosted glass effects and vibrant indigo-purple gradients.
- 📱 **Fully Responsive**: Optimized for all devices with adaptive animation performance.
- 🤖 **AI Chatbot**: Integrated assistant to help visitors navigate the portfolio.
- 🏗️ **Project Showcase**: 3D perspective cards with parallax effects and category filtering.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or bun

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/nehilghetia/nehil-s-dev-showcase.git
   ```
2. Navigate to the project directory:
   ```bash
   cd nehil-s-dev-showcase
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production
```bash
npm run build
```

## 📂 Project Structure
- `src/components`: UI components (Hero, Skills, Projects, etc.)
- `src/components/ui`: shadcn/ui base components
- `src/lib`: Utility functions and configuration
- `public`: Static assets (images, icons, etc.)
